English to Runes
======================
A command line tool to switch English letters to Runes letters or vice versa.
Written in Python3.
I just a newbie in Python, so the code is rookie-like.

[Usage]
------
    $python3 Runes.py -e|-d [file]                                                 
         -e: English to Runes                                                                          
         -d: Runes to English                                                                          
         use stdin when no file input
